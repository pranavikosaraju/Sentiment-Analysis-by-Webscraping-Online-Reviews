# Sentiment-Analysis-by-Webscraping-Online-Reviews
The data for this project acquired by Webscraping using the Python Library â€“ BeautifulSoup. Reviews of Iphone 12 Mini is webscraped from an e-Commerce website. The reviews are converted into a Pandas DataFrame wherein Null values and Duplicates are removed. The reviews are then cleaned by removing stopwords, special characters etc using the NLTK library.
